# 📉 Customer Churn Prediction using Machine Learning

## 🎯 Objective

The goal of this project is to predict which customers are likely to leave (churn) so that businesses can take proactive actions to retain them.

---

## 📂 Dataset

* Dataset: Telco Customer Churn Dataset
* Includes customer demographics, services, charges, and churn status.

---

## 🧠 Approach & Methodology

### 1. Data Cleaning

* Handled missing values
* Converted data types appropriately
* Removed irrelevant columns

### 2. Data Preprocessing

* Applied Label Encoding to convert categorical data into numeric form

### 3. Train-Test Split

* Split dataset into training and testing sets
* Ensures proper evaluation of model performance

### 4. Model Building

* Logistic Regression (baseline model)
* Random Forest (improved performance)

### 5. Model Evaluation

* Accuracy Score
* Confusion Matrix

---

## 📊 Key Insights

* Customers with high monthly charges are more likely to churn
* New customers have a higher churn probability
* Customers without support services (e.g., TechSupport) churn more

---

## 💡 Business Recommendations

* Provide better customer support services
* Improve onboarding experience for new customers
* Offer discounts or benefits to high-paying customers

---

## 🛠️ Tools & Technologies

* Python
* Pandas, NumPy
* Scikit-learn (Logistic Regression, Random Forest)
* Matplotlib / Seaborn

---

## 🚀 Conclusion

Churn prediction helps businesses reduce customer loss by identifying at-risk customers early and taking preventive measures.
